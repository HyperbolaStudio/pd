Introduction

What is this

Plastic Design provides a set of cross-platform design interfaces(including iOS, Android, desktop and web) with documentation available. Plastic Design defines a uniform set of user-element interactions, movements of elements and visual effects. This document defines the specifications of Plastic Design and how to use the interfaces required by Plastic Design.

Goals

Our purpose

We challenge to combine brilliant design principles and innovative technology.We create a simple underlying system that allows cross-platform developing.We have a unique principle of the motion of elements and the input method like touch, voice, mouse, and keyboard of users,as well as stipulate how the user's actions affect the movement of the element.
