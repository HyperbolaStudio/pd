Introduction

What is this

Plastic Design provides a set of cross-platform design interfaces(including iOS, Android, desktop and web) with documentation available. Plastic Design defines a uniform set of user-element interactions, movements of elements and visual effects. This document defines the specifications that Plastic Design enforces and how to use the interfaces required by Plastic Design.
